# Depix Examples

Standalone reference examples showing how to use the `@depix/*` packages.

## Examples

| File | Description |
|------|-------------|
| `01-basic-react.tsx` | Render a DSL diagram with `<DepixCanvas>` |
| `02-tiptap-integration.ts` | Serialize/parse Depix blocks in Markdown |
| `03-programmatic-ir.ts` | Build a DepixIR document programmatically |
| `04-custom-theme.ts` | Create and apply a custom theme |

## Prerequisites

```bash
pnpm install
pnpm build
```

## Usage

These files are reference code — copy the patterns into your own project.
To see a live demo, run:

```bash
pnpm --filter @depix/demo dev
```
